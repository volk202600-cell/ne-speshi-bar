НЕ СПЕШИ БАР — Локальний сайт (Frontend + Backend)
=============================================

Що всередині:
- backend/ - Node.js сервер (scrape меню та відправлення в Telegram)
- frontend/ - статична сторінка (index.html, app.js)
- start.bat - скрипт для Windows, який встановлює залежності і запускає сервер

Перед запуском:
1) Скопіюйте файл backend/.env.example -> backend/.env і заповніть:
   MENU_SOURCE_URL=https://menu.ps.me/s5_XOHGKU7w
   TELEGRAM_BOT_TOKEN=ВАШ_ТОКЕН_БОТА
   TELEGRAM_CHAT_ID=ВАШ_CHAT_ID

   Якщо не хочете підключати Telegram зараз, лишіть TELEGRAM_BOT_TOKEN і TELEGRAM_CHAT_ID порожніми — замовлення все одно будуть прийматись, але повідомлення в Telegram не надсилатимуться.

Як запустити (Windows):
1) Розпакуйте архів у папку, наприклад C:
e_speshi_bar
2) Відкрийте папку, двічі клацніть start.bat
   Скрипт:
   - встановить залежності в backend (npm install)
   - запустить backend на порту 4000
3) Відкрийте браузер: http://localhost:4000

Як тестувати:
- Додайте товари в кошик
- Заповніть ім'я та телефон
- Натисніть "Підтвердити замовлення"
- Побачите повідомлення "Ваше замовлення прийняте"
- Якщо в backend/.env вказаний токен і chat_id, повідомлення також надійде в Telegram

Примітка про scraping:
- Сервер намагається автоматично витягнути назви та ціни зі сторінки MENU_SOURCE_URL.
  Результат залежить від структури сайту. Якщо дані не будуть коректні, ви можете змінити menu у backend/data/menu.json вручну.

Питання / допомога:
- Я можу допомогти вставити токен безпечно або налаштувати хостинг.
